Category Blindness

This is kind of like saying, "If all you have is a hammer everything looks like a nail", but it's more insidious than that because if you've tried a lot of tools, but only in one category, then you're experiencing "Category Blindness".

These are different categories of tools, but can fill the same Job to Be done.

Some examples are when people say:

  • "All programming languages are the same" but have never tried Functional Programming or Logic Programming.
  • "Slack is much better than E-mail" but have only used Synchronous Communication tools, never Asynchronous Communication tools.
  • Etc...
