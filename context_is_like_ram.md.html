Context is like RAM

created: 2020-11-11T03:09:01-10:00 modified: 2020-11-11T03:16:58-10:00 title: Context is like RAM

When knowledge rely on context you're assuming that that context will never go away. This is usually false because that context is usually Tribal Knowledge at best, only stored in one person's head at worst.

The problem is, just like RAM, using context is faster in the now. Slack understands this. It's addictive to trade the future for now.

Documentation is like a Hard Drive; Slower, yet far less dangerous to lose something important.
