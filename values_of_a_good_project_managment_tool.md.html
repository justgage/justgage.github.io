Values of a good Project Managment Tool

  1. Prevent communication Fragmentation: when a person is remote means they can't talk around a product, they have to talk in it, because of this it must naturally organize information
  2. Transperency: You can know what a team is working on and how it's going because it's far less visible when you're remote.
  3. Information Stability: Information needs to stay in the same place, if it's hard to find it will be duplicated.
