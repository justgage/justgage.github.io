Jetbrains Spaces

ref: https://www.jetbrains.com/space/

Seems to be something between Slack, Gitlab and Basecamp.

Things I love:

  • the onroadmap.com like vacation timeline
  • A personalized to-do lists like BC4's "My Place" (mentioned in) Ryan Singer's newsletter.
  • the code commenting tool
  • How it's like Basecamp's an all-in-one solution
  • How it integrates with Git
  • Blog posts which are like how Basecamp replace meetings with the message board
  • Chats are much more thread based!
  • Polls with lots of options:
