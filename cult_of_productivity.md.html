Cult of productivity

Being productive means you increase the speed of which you are producing some artifact. It seems that speed can never be bad right? However speed is often a distraction for progress. you think a lot about how fast we are climbing the ladder not where the ladder is against the right wall. I'm trying to type code faster but we don't think about if the code is the what we should be writing. Think about how fast tickets get through the pipeline and not work all the tickets add up to.
