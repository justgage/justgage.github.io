going slow to go fast

Relates to the idea that doing things slowly now, often helps you go faster in the future. documentation is a great example of this, you invest some time to help answer a question that someone in the future might have. Stack Overflow is a good example of this. It's value is compounding, whereas Chat is like Renting, Documentation is like Owning

Kind of the opposite of: Doing things the fun way.
