Agile is about productivity, Basecamp is about effectiveness

Agile is about getting as many things done as quick as possible (pipeline efficiency). However, it lacks a great way to see if the things people are doing are actually the right things without a deep look into every ticket.

Focus on how fast you can get things through the funnel instead of trying to guard the top of the funnel very judiciously.

This is a related to the idea that code is a liability. So trying to write this code if you can avoid it then write code that is not going to be useful to you in the future.
